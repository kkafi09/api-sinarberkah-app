
# API Sinarberkah Plat Ornament

API for Sinarberkah Plat Ornament app. This service is for user, gallery, and article







## API Reference

- User
```http
  GET /api/user/v1
```

- Gallery 
```http
  GET /api/gallery/v1
```

- Article 
```http
  GET /api/article/v1
```
